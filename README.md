# 价值量模型关联数据基准文件

## corr_data.json
存储2年(490个交易日)的历史关联数据，供网页相关性图表使用。

| 字段 | 说明 | 数据来源 | 更新频率 |
|------|------|---------|---------|
| ccfi | 中国出口集装箱运价指数 | 上海航运交易所(周度插值) | 周度 |
| wti | WTI原油期货价格(美元/桶) | EIA(周度插值至日) | 周度 |
| methionine | 固体蛋氨酸均价(元/kg) | 饲料行业信息网(周度插值) | 周度 |
| ashare | A股全市场成交额(万亿元) | 东方财富(日度) | 每日 |
| cement | 全国水泥指数CEMPI(2009=100) | ccement.com(周度插值) | 周度 |
| dates | 交易日列表 | 通达信 | 每日 |

## 数据更新方式
每日运行时，只查询最新数据追加到数组末尾，不再重新生成全部历史数据。
